HoverIntent

Delayed hover event handlers.

Only trigger the event handler when the user intents to hover the element.

Hover the diffent lines below to see the effect.

ngMouseOver, immediate.

ngHoverintent, 500ms (default delay).

ngHoverintent, 5 seconds delay.

ngHoverintent, 5 seconds delay. Delay reset when clicked.
